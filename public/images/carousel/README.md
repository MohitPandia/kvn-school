# Carousel images

Add your school photos here (staff, students, campus). Use any names (e.g. `1.jpg`, `2.jpg`, `slide-1.jpg`).

Then list the filenames in **`src/lib/gallery.ts`** in the `carouselImages` array so they appear in the landing carousel. Example:

```ts
// In src/lib/gallery.ts, add paths like:
{ src: "/images/carousel/1.jpg", alt: "Description" },
```
